PROPOSITO DEL CURSO


El proposito del curso es tener las bases necesarias para poder desarrollar y crear paginas web basicas, de esa manera como estudiantes se lográn tener las capacidades para desarrollar todo tipo de sitio web para el entorno laboral

INTRODUCCIÓN


En el desarrollo de esta actividad se encontrará 3 temas el cual tiene sus 3 recursos el cual ayuda a mejorar el conocimiento previo y dar más avance en el desarrollo del curso, de la misma manera información del desarrollador de la pagina, usando cada una de las fuentes correspondientes para poder garantizar un aprendizaje correcto y profesional. Podrá navegar en la amable interfaz la cual tiene colores que ayudan a los ojos y no ser hostigantes para el mismo. Con el pie de pagina en cada una de las pagina podrá ver el lugar que adquiere el estudiante los conocimiento, como a continuación al lado derecho.
