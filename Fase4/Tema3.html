Descripción Enlaces


Los enlaces se pueden encontrar en cualquier lugar dentro de internet, estos permiten la conexión entre varias paginas o para acceder a una pagina web. La etiqueta para poder crear los enlaces es "a" dentro de eso se genera la siguiente "a href ="..." ", dentro de esas comillas se podra colocar enlace que uno desee. Ejemplo:
