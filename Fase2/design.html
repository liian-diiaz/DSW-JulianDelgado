Proposito del curso


El proposito del curso es tener las bases necesarias para poder desarrollar y crear paginas web basicas, de esa manera como estudiantes se lográn tener las capacidades para desarrollar todo tipo de sitio web para el entorno laboral

Flex Box


Es una de las propiedades de CSS display, que nos permite colocarle mejor diseño a nuestra pagina web. El principal objetivo detras del diseño flexible es dar la capacidad de poder alterar el ancho como el alto de los articulos que uno ha indicado. Es un diseño apropiado especialmente para aplicaciones o diseños a pequeña escala. Al trabajar con el Flexbox se necesita pensar en dos terminos de ejes, el eje principal y el eje cruzado.

Eje principal

Se define con flex-direction el cual tiene 4 valores que se le pueden asignar.

  1. row
  2. row-reverse
  3. column
  4. column-reverse

Con ese ejemplo de desea mostrar las siguientes propiedades de Flexbox: Con justify-content es una propiedad que nos ayuda alinear los items que deseamos arreglar. Con eso se tienen los siguientes valores para poder realizar los adjuntes.

  • space-evenly
  • flex-start
  • flex-end
  • center
  • space-around
  • space-between
